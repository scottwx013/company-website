## 域名配置

**目标域名**: www.1gift.cn

### DNS 配置步骤

1. 登录你的域名管理后台（如阿里云、腾讯云、DNSPod）
2. 找到 DNS 解析设置
3. 添加 CNAME 记录：
   - 主机记录: `www`
   - 记录类型: `CNAME`
   - 记录值: `cname.vercel-dns.com`
   - TTL: 默认（600秒）

### 部署步骤

1. 我部署到 Vercel 生产环境
2. 在 Vercel 后台添加自定义域名 `www.1gift.cn`
3. 你配置 DNS CNAME 记录
4. 等待 DNS 生效（通常 5-30 分钟）
